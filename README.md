Project Overview

A simple Calculator Application developed using Python.
This project performs basic arithmetic operations and helps understand Python concepts like functions, user input handling, conditional statements, and loops.

Features
Addition
Subtraction
Multiplication
Division
Handles user input
Displays calculation results

Concepts Used:
Python Variables
Functions
Conditional Statements (if, elif, else)
Loops (while)
Exception Handling (try, except)
User Input (input())

How to Run This Project
Step 1:

Install Python

Step 2:

Clone the repository

git clone <your-github-link>
Step 3:

Move into project folder

cd calculator-application
Step 4:

Run the file

python main.py

Sample Output:

Simple Calculator

Enter First Number: 10
Enter Operator (+,-,*,/): *
Enter Second Number: 5

Result = 50
